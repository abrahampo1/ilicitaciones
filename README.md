# ILicitaciones

ILicitaciones es una aplicación web desarrollada en Laravel 12 para la consulta, análisis y seguimiento de licitaciones públicas. El proyecto busca centralizar información relevante de procesos de contratación pública y facilitar su análisis para apoyar la toma de decisiones.

Está orientado a empresas, analistas, investigadores y cualquier persona interesada en monitorear y estudiar licitaciones del sector público.
